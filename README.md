# Spectral Graph Toolkit

A Python toolkit for computing **structural** and **spectral invariants** of graphs.

This project is designed for students, researchers, and practitioners interested in **graph theory**, **spectral graph theory**, and **network analysis**.

---

## 🚀 Features

### Structural Invariants
- Number of vertices and edges
- Degree sequence (min, max, average)
- Connected components
- Diameter and radius
- Independence number (exact for small graphs)
- Clique number (exact for small graphs)

### Spectral Invariants
- Adjacency eigenvalues and eigenvectors
- Laplacian eigenvalues and eigenvectors
- Signless Laplacian spectrum
- Spectral radius
- Algebraic connectivity
- Graph energy
- Number of spanning trees (Matrix Tree Theorem)

### Visualization
- Graph drawing
- Adjacency matrix heatmap
- Spectrum plots (Adjacency & Laplacian)

---

## 📌 Motivation

Spectral graph theory provides powerful tools to understand the global structure of graphs through eigenvalues and eigenvectors.

This toolkit aims to:
- bridge theory and computation,
- provide intuitive insights into graph spectra,
- serve as a lightweight platform for experimentation and teaching.
